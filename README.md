## About the project 
<p align="left">
A Todo List application built with React that allows users to easily manage tasks:
add, edit, delete, filter, and save data in the browser.
</p>

  ## Installation 
   #### 1-Clone the project
     ```bash
    git clone https://github.com/zaineb-123/to_do_list_app.git
     ```
  #### 2-acces to the folder
     ```bash
    cd to_do_list_app.git
     ```
  
  #### 3-Install NPM packages
     ```bash
     npm install
     ```
  #### 4-run the app
     ```bash
      npm run dev
     ```
